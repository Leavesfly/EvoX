# 框架层 (Framework Layer)

框架层提供多智能体协同框架和高级交互模式。

## 模块列表

| 模块 | 说明 | 依赖 |
|------|------|------|
| **evox-frameworks** | 多智能体框架，提供辩论系统等高级协同模式 | evox-core, evox-agents |

## 设计原则

- **协同优先**: 专注于多智能体协同场景
- **模式封装**: 封装常见的协同模式
- **易于使用**: 提供简单的 API 接口
- **可扩展**: 支持自定义协同模式

## 依赖关系

```
evox-frameworks ──┬──> evox-core
                  └──> evox-agents
```

## 典型场景

- **多智能体辩论**: 多个 Agent 通过辩论达成共识
- **协同决策**: 多个专家 Agent 协同完成复杂任务
- **任务分配**: 智能分配任务给不同的 Agent
